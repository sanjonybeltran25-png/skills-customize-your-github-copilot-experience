# 📘 Assignment: Python Testing and Debugging

## 🎯 Objective

Learn how to write automated tests with `pytest`, validate code behavior, and debug Python functions using clear error handling.

## 📝 Tasks

### 🛠️ Add Unit Tests

#### Description
Write automated tests for the sample utility functions using `pytest`.

#### Requirements
Completed program should:

- Include at least 3 unit tests using `pytest`
- Test normal behavior for arithmetic and string formatting functions
- Test edge cases such as division by zero or invalid inputs
- Use assertions to verify expected results

### 🛠️ Improve Error Handling

#### Description
Update the sample functions so they handle invalid input and return clear error messages.

#### Requirements
Completed program should:

- Validate input for `safe_divide()` and raise a `ValueError` for division by zero
- Ensure text formatting functions handle whitespace correctly
- Keep functions easy to test and debug

### 🛠️ Run and Debug the Tests

#### Description
Run the test suite, identify failing behavior, and fix any bugs in the code.

#### Requirements
Completed program should:

- Run the tests successfully with `pytest`
- Fix any failing code paths discovered by the tests
- Confirm the final solution returns correct values and raises expected errors
